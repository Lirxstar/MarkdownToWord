A browser-based Markdown to Word (`.docx`) converter with support for LaTeX formulas, local images, and Chinese/English interface switching.

## Features

* Convert Markdown content to `.docx`
* Upload a `.md` file or paste Markdown text directly
* Preserve common Markdown structures:

  * headings
  * paragraphs
  * lists
  * tables
  * blockquotes
  * code blocks
* Support LaTeX formulas:

  * inline formulas: `$E = mc^2$`
  * display formulas: `$$ ... $$`
* Support local images referenced in Markdown
* Chinese and English interface switching
* Browser-based conversion
* No backend server required

## Usage

1. Open the website.
2. Upload a Markdown file or paste Markdown content into the editor.
3. Upload local images if the Markdown file references local image paths.
4. Click the convert button.
5. Download the generated Word `.docx` file.
