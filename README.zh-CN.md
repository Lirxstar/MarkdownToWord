# Markdown 转 Word 工具

Please use the link to access directly: https://lirxstar.github.io/MarkdownToWord/
请使用这个链接直接访问：https://lirxstar.github.io/MarkdownToWord/

[English README](README.md)

这是一个基于浏览器运行的 Markdown 转 Word（`.docx`）工具，支持 LaTeX 公式、本地图片，以及中英文界面切换。

## 功能

* 将 Markdown 内容转换为 `.docx`
* 支持上传 `.md` 文件
* 支持直接粘贴 Markdown 内容
* 支持常见 Markdown 结构：

  * 标题
  * 段落
  * 列表
  * 表格
  * 引用
  * 代码块
* 支持 LaTeX 公式：

  * 行内公式：`$E = mc^2$`
  * 独立公式：`$$ ... $$`
* 支持 Markdown 中引用的本地图片
* 支持中文和英文界面切换
* 在浏览器中完成转换
* 不需要后端服务器

## 使用方法

1. 打开网站。
2. 上传 Markdown 文件，或直接粘贴 Markdown 内容。
3. 如果 Markdown 中引用了本地图片，请同时上传对应图片。
4. 点击转换按钮。
5. 下载生成的 Word `.docx` 文件。
